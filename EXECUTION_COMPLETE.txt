HOW TO ACCESS THIS RELEASE
================================================================================

1. Clone Repository
   git clone https://github.com/harikrshm/Mutual-fund-FAQ-Assistant_RAG.git

2. Checkout Release
   cd Mutual-fund-FAQ-Assistant_RAG
   git checkout v0.1-mf-faq

3. Setup Python Environment
   python -m venv .venv
   source .venv/bin/activate  # Windows: .\.venv\Scripts\Activate.ps1

4. Install Dependencies
   pip install -r requirements.txt

5. Run Test Suite
   pytest -v                          # All tests
   pytest --cov=src tests/           # With coverage
   pytest tests/test_faq_logic.py -v # Specific module

6. Start API Server
   python src/api/server.py
   # API available at http://localhost:5000

7. Test with Sample Queries
   # See sample_faqs/sample_faq.csv for 10 pre-tested queries
   # Or use examples in README.md

8. Review Documentation
   cat README.md                         # Setup guide
   cat COMPLIANCE_CHECKLIST.md          # Compliance verification
   cat SECTION_6_EXECUTION_SUMMARY.md   # Detailed report
